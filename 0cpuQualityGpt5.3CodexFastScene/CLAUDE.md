# Scene-Focused Fast CPU x265 Encoding Presets

## Overview

This folder contains scene-focused fast CPU x265 encoding presets that adapt encoding parameters based on scene detection and content analysis for optimal speed and quality balance.

## Use Cases

- Scene-adaptive video encoding
- Content-aware fast transcoding
- Dynamic quality adjustment based on video content
- Intelligent scene detection encoding

## Features

- **Scene Detection**: Adaptive encoding parameters based on detected scenes
- **Content Analysis**: Intelligent quality adjustment per scene
- **Speed**: Fast encoding times with scene-aware optimization
- **Quality**: Dynamic quality preservation matching scene complexity

## Encoding Characteristics

- **Codec**: H.265/HEVC (libx265)
- **Bit Depth**: 10-bit
- **Mode**: Scene-aware CPU encoding
- **Optimization**: Scene detection and adaptive bitrate allocation
- **Audio Codec**: AAC (libfdk_aac)

## Performance Benefits

- Optimizes encoding time for complex scenes
- Allocates bitrate more efficiently across scenes
- Maintains visual quality on important scenes
- Reduces unnecessary quality overhead on simple scenes

## When to Use

Use these presets when you want fast encoding with scene-intelligent quality optimization, particularly for content with varying scene complexity.
