# H.265 10-Bit NVIDIA NVENC Encoding Presets

## Overview

This folder contains H.265/HEVC video encoding presets using NVIDIA NVENC GPU acceleration with 10-bit encoding depth. These presets provide modern codec efficiency with GPU-accelerated performance.

## Use Cases

- GPU-accelerated H.265 encoding workflows
- Modern device compatibility (H.265 support)
- 10-bit color depth for better gradation
- Real-time encoding on NVIDIA systems
- High-performance batch video processing

## Encoding Characteristics

- **Codec**: H.265/HEVC
- **Bit Depth**: 10-bit (better color precision)
- **Acceleration**: NVIDIA NVENC GPU hardware encoding
- **Speed**: Very fast GPU-accelerated encoding
- **Compatibility**: Modern devices (not all legacy devices)
- **Audio Codec**: AAC (libfdk_aac)

## Hardware Requirements

- NVIDIA GPU with NVENC H.265 support:
  - GeForce GTX 960/970 and newer
  - RTX series for optimal performance
  - Professional Tesla/Quadro cards

## Benefits

- **Speed**: Fast GPU-accelerated H.265 encoding
- **Compression**: 50% better efficiency than H.264
- **Quality**: 10-bit depth for superior color gradation
- **Efficiency**: Offloads to GPU, minimal CPU usage

## Performance Profile

- Encoding speed: Very fast (GPU accelerated)
- Quality level: High (excellent compression efficiency)
- File size: 40-50% smaller than H.264 for same quality
- CPU usage: Minimal (GPU handles encoding)

## Color Precision

10-bit encoding provides:
- Better color gradation (4K colors per channel vs 256)
- Reduced banding artifacts
- Professional quality color reproduction
- HDR content preparation

## Device Compatibility

- Modern TVs, smartphones, and players
- Some older/legacy devices may not support H.265
- Desktop browsers have improving H.265 support

## Ideal For

- Modern device ecosystem encoding
- Streaming services and distribution
- 4K content and HDR workflows
- GPU-accelerated production pipelines
- High-quality archival with space efficiency
