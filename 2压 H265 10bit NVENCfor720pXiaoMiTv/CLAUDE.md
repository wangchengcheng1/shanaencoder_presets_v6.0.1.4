# H.265 10-Bit NVENC for 720p Xiaomi TV

## Overview

This folder contains specialized H.265/HEVC encoding presets using NVIDIA NVENC optimized specifically for 720p playback on Xiaomi television sets.

## Target Device

- **Xiaomi TV** models with H.265 support
- **Resolution**: 720p (1280x720)
- **Optimization**: Tuned for Xiaomi hardware decoders
- **Quality**: Optimized for 720p display characteristics

## Use Cases

- Xiaomi smart TV content preparation
- 720p video distribution for Xiaomi devices
- Optimized encoding for specific TV hardware
- Compatibility-focused video encoding

- **Bit Depth**: 10-bit
- **Resolution**: Optimized for 720p
- **Acceleration**: NVIDIA NVENC GPU
- **Target Device**: Xiaomi TV hardware
- **Audio Codec**: AAC (libfdk_aac)
