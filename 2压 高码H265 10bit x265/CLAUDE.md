# High Bitrate H.265 10-Bit x265 Encoding Presets

## Overview

This folder contains high-bitrate H.265/HEVC encoding presets using x265 CPU encoder with 10-bit encoding depth. These presets prioritize maximum quality with higher bitrates for professional and archival use.

## Use Cases

- Professional video archival
- Maximum quality video encoding
- Studio-grade video production
- High-bitrate streaming for premium platforms
- 4K and ultra-high-quality content

## Encoding Characteristics

- **Codec**: H.265/HEVC
- **Bit Depth**: 10-bit (professional color precision)
- **Bitrate**: Higher (prioritizes quality over file size)
- **Encoder**: x265 (CPU-based software)
- **Quality Level**: Maximum quality
- **Audio Codec**: AAC (libfdk_aac) at high bitrates

## Features

- **Quality**: Highest quality H.265 encoding
