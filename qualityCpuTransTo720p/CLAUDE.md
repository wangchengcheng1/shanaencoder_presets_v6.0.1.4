# CPU Quality Transcoding to 720p

## Overview

This folder contains CPU-based quality encoding presets that transcode video to 720p (HD) resolution. These presets combine video quality optimization with resolution scaling to 720p.

## Use Cases

- Downscaling to 720p HD resolution
- Mobile device compatible encoding
- Streaming optimization to 720p
- Bandwidth-efficient distribution
- Smaller file size with good quality

## Encoding Characteristics

- **Resolution**: 720p (1280x720)
- **Quality Level**: Optimized CPU quality presets
- **Encoder**: x265 CPU-based
- **Bit Depth**: 10-bit
- **Scaling**: High-quality image downscaling
- **Audio Codec**: AAC (libfdk_aac)

## Resolution Details

- **HD Resolution**: 1280x720 pixels
- **Aspect Ratio**: Standard 16:9
- **Frame Rate**: Maintained from source
- **Size**: Smaller than 1080p, suitable for streaming

## Scaling Quality
