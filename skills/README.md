# Skills Hub for ShanaEncoder Presets

This folder contains project-local skills for faster preset iteration.

## Available skills

- `shanaencoder-preset-maintainer`: safe bulk maintenance for XML presets.
- `shanaencoder-x265-lab`: scene-focused x265 tuning presets and guardrails.
- `shanaencoder-preset-arena`: run A/B/C "battles" and pick winners by scorecard.
- `shanaencoder-preset-quest-board`: turn tuning into mission cards (size/speed/quality modes).
- `shanaencoder-regression-guard`: prevent accidental mass-regressions before rollout.

## How to use

1. Pick one skill based on current goal.
2. Keep stable presets unchanged; create branch presets for experiments.
3. Validate on short clips first, then apply in batch.
