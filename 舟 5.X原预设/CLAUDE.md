# ShanaEncoder 5.X Original Presets (Legacy)

## Overview

This folder contains original preset configurations from ShanaEncoder version 5.X. These are legacy presets preserved for backward compatibility and historical reference.

## Purpose

- **Legacy Support**: Maintain compatibility with older 5.X version workflows
- **Reference**: Historical preset configurations
- **Migration**: Reference for upgrading to newer versions
- **Archive**: Preserve original preset designs

## Historical Context

- **Version**: ShanaEncoder 5.X
- **Status**: Legacy/superseded by 6.0
- **Usage**: Primarily for backward compatibility
- **Maintenance**: No active development

## Important Notes
